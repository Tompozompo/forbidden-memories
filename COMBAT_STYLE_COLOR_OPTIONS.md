# Combat Style & Element Color Schemes

## Concept: Attributes (Defense) + Combat Styles (Offense)

**Visual Design Idea:**
- **Card Background/Base Color** = Attribute (what the monster IS)
- **Card Border Color** = Combat Style (how the monster ATTACKS)
- Each card has: 1 Attribute color + 1-2 Combat Style border colors

## Implementation Notes

### CSS Example (Dual Border)
```css
.card {
  background-color: var(--attribute-color); /* Base: LIGHT = gold */
  border: 3px solid var(--primary-style); /* Radiance = gold */
  box-shadow: 0 0 0 3px var(--secondary-style); /* Frost = ice blue */
}
```

### Color Overlap Strategy
When attribute and style colors are similar (e.g., FIRE attribute + Flame style):
1. **Option A:** Use different shades (lighter border on darker base)
2. **Option B:** Add glow/shadow effect to differentiate
3. **Option C:** Use gradient between base and border

---

## Final Recommendation

**Naming:** Stick with **Current** (Flame, Torrent, etc.) - clear and universal
**Colors:** Use **Vibrant Spectrum** for borders, **Primary Colors** for attributes
**Differentiation:** Attribute = solid fill, Combat Style = thick border (3-5px)
**Accessibility:** Always include icon + text label + color

This gives you 10 distinct combat styles with appealing, recognizable colors that work well with the 6 attribute colors.
